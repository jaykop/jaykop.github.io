## 개요
* 크롬 엑스텐션 중 하나인 Momentum Cloning 프로젝트
* JavaScript 이용한 TODO LIST 웹 어플리케이션  

## 구현 기능
- 랜덤으로 바뀌는 배경화면
- 리스트 추가/삭제 가능
- 현재 위치와 위치 기반 시간 기능
- 유저의 이름 입력 가능
